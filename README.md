# Financial Statement Analysis Dashboard

An interactive Streamlit dashboard for analyzing multi-year corporate financial statements. The application computes key financial ratios, growth metrics, CAGR, and a weighted investment score while providing visual insights into company performance.

## Features

* Financial ratio analysis (ROE, ROA, Net Margin, Debt-to-Equity, Asset Turnover, Earnings Quality)
* Revenue & Profit Growth analysis
* Revenue & Profit CAGR calculation
* Investment Scoring Model for company ranking
* Interactive KPI dashboard and financial trend visualizations
* Company-wise filtering and comparative analysis

## Tech Stack

* Python
* Pandas
* NumPy
* Streamlit
* Matplotlib
* Excel

## Run Locally

```bash
git clone https://github.com/your-username/Financial-Statement-Analysis.git
cd Financial-Statement-Analysis
pip install -r requirements.txt
streamlit run app.py
```
